Данный проект был создан, чтобы люди могли подавать объявления о продаже машин, так же искать для покупки. Проект был создан с использованием библиотеки ReactJS (Create react app)

Для полноценной работы приложения, вам необходимо также скачать и параллельно запустить API к приложению, которое можете найти по ссылке https://github.com/gutsyheart-talgat/final-api

Скопируйте проект
https://github.com/gutsyheart-talgat/final-project.git

Для установки пакетов, перед запуском используйте 
npm install

Запуск проекта
npm start

Для сборки запустить 
npm run build

Зарегистрируйтесь и попробуйте подать свое объявление.
